# Generate Epic Transition Report

## Purpose

Generate the final **Epic Transition Report** for the completed Epic.

---

## Instructions

- Read the entire conversation before generating the report.
- Use only information available in the current Epic.
- Do not invent information.
- If information is missing, explicitly state **"Not defined"**.
- Write the report entirely in **English**.
- Use **GitHub Flavored Markdown (GFM)**.
- Follow the template:

```
docs/templates/Epic_Transition_Report_Template.md
```

Return only the completed Markdown document.
